**Project Description**

This game was developed using Unity 2022.3.21f1 for Windows.

The game is set in a cemetery maze, and the main character is the Grave Digger. Within the maze, there are two keys that the player must collect. After obtaining both keys, the player needs to reach the grave containing the treasure to win the game.

Throughout the maze, ghosts patrol specific points. When they spot the main character, they begin to chase them. If the character leaves the ghosts' detection area, the ghosts will search the last seen location for a while. If they do not find the character there, they return to the nearest patrol point and resume their regular patrol.

The ghosts have four states:

Patrol – following predefined paths
Chase – pursuing the main character
Search – looking for the character at the last seen location
Catch – capturing the character, which results in Game Over and restarts the game

Pathfinding for the ghosts is implemented using the A algorithm*.
